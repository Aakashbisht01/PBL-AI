This project is an AI-powered assistant designed to help students write better research papers. It leverages natural language processing (NLP) and machine learning to assist in various stages of academic writing — from brainstorming topics to polishing the final draft.

✨ Features
🔍 Topic Suggestion based on field of interest

🗂️ Outline Generator for structured writing

📚 Reference and Citation Finder using academic databases

✍️ Grammar & Style Enhancer powered by NLP

🧠 Plagiarism Checker (optional, API-based)

💬 Chat-based Writing Assistant to guide students in real-time

🛠️ Technologies Used
Python (Flask / FastAPI)

OpenAI GPT API (or any LLM)

spaCy / NLTK for text processing

Semantic Scholar API or CrossRef API (for references)

React (optional frontend)

MongoDB
